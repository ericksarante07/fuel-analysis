# Fuel Price Analysis

Proyecto para limpieza, segmentación y análisis de datos de combustibles
(diésel y gasolina).

## Estructura

- `src/`: lógica de procesamiento reutilizable
- `analysis/`: notebooks de análisis exploratorio
- `data/`: datos crudos y procesados
- `main.py`: script ejecutable

## Instalación

```bash
pip install -r requirements.txt
